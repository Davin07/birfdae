# Contributing to Birthday Reminder App

First off, thank you for considering contributing to the Birthday Reminder App! It's people like you that make open source such a great place to learn, inspire, and create.

## How Can I Contribute?

### Reporting Bugs
- Use the GitHub Issue Tracker.
- Check if the bug has already been reported.
- If not, create a new issue with a clear title and description, steps to reproduce, and any relevant logs or screenshots.

### Suggesting Enhancements
- Enhancement suggestions are tracked as GitHub issues.
- Provide a clear and detailed description of the proposed enhancement.
- Explain why this enhancement would be useful to most Birthday Reminder App users.

### Pull Requests
1. Fork the repository.
2. Create a new branch for your feature or bug fix.
3. Make your changes, ensuring you follow the project's code style.
4. Add tests for your changes.
5. Ensure all tests pass.
6. Submit a pull request with a clear description of the changes.

## Code Style
Please follow the official Kotlin coding conventions as specified in the project configuration.
