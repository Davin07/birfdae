# Tech Stack - Birthday Reminder App

## Core Technologies
*   **Language:** Kotlin (v1.9.10) - Primary language for Android development.
*   **UI Framework:** Jetpack Compose with Material Design 3 (Material You).
*   **Architecture:** Clean Architecture with separate Data, Domain, and UI layers.
*   **Dependency Injection:** Hilt (Dagger) for managing component lifecycles and dependencies.
*   **Local Database:** Room (v2.6.1) for persistent SQLite storage.
*   **Background Tasks:** WorkManager (v2.9.0) for reliable notification scheduling and background jobs.

## Build & Environment
*   **Build System:** Gradle (Kotlin DSL).
*   **Java Version:** Java 17 (JVM Target 17).
*   **Minimum SDK:** 21 (Android 5.0).
*   **Target SDK:** 34 (Android 14).

## Libraries & Tools
*   **Navigation:** Navigation Compose for screen routing.
*   **Logging:** Timber for enhanced logging.
*   **Serialization:** Kotlinx Serialization for JSON processing.
*   **Testing:** JUnit 4, Mockito, Turbine, and Espresso for comprehensive test coverage.
